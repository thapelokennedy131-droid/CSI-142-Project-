## Concept Note
Our system manages accounts, cashflow, and calculate the accumulated money in the account records
for the motshelo members. It allows members to create accounts,
track cashflow, and track financial history.

## Repository
This project is built using Java with OOP principles
including inheritance, composition, polymorphism, abstraction, collections, packages, robustness,git workflow and encapsulation.
